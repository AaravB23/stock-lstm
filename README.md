# Stock Long Short-Term Memory

## Purpose
This is for the MMC Scholar's program for an eventual machine learning class.
It trains on historical stock data using a stacked LSTM neural network.

## Usage
Download a stock CSV with `date` and `close` columns, e.g. this [Kaggle](https://www.kaggle.com/datasets/camnugent/sandp500) dataset.
Run main.py and provide the CSV path.
